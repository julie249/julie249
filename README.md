An AI-powered travel planning platform that generates personalized trip itineraries based on user preferences.

**Tech Stack:** React.js, Vite, Tailwind CSS, Node.js, Express.js, OpenAI API  

**Key Features:**
- AI-generated travel itineraries  
- Destination-based trip planning  
- PDF itinerary download  
- Saved trips using local storage  
- Clean and responsive UI
